高级功能示例

系统托盘

系统托盘图标已创建，可以在系统托盘区域找到它。

点击系统托盘图标可以显示菜单，包括显示主窗口、功能菜单和退出选项。

全局快捷键

已注册以下全局快捷键：

  • Ctrl+Shift+T 显示系统托盘页面
  • Ctrl+Shift+S 显示全局快捷键页面
  • Ctrl+Shift+C 显示剪贴板页面
  • Ctrl+Shift+P 显示屏幕捕获页面
  • Ctrl+Shift+U 显示自动更新页面
  • Ctrl+Shift+Q 退出应用

剪贴板

剪贴板功能允许你读取和写入剪贴板的文本、HTML 和图片。

屏幕捕获

屏幕捕获功能允许你捕获屏幕截图和录制屏幕。

自动更新

自动更新功能允许你检查、下载和安装应用更新。
