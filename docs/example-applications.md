# NativePHP/ThinkPHP 示例应用

## 1. 任务管理应用

一个简单的任务管理应用，展示基本的 CRUD 操作和 UI 交互。

### 功能特点
- 创建、编辑、删除任务
- 任务分类和标签
- 任务提醒和通知
- 数据本地存储
- 数据导入导出

### 技术要点
- 使用 Window 和 Menu 组件
- 使用 Notification 组件
- 使用 Database 组件
- 使用 FileSystem 组件

### 实现步骤
1. 创建基本应用结构
2. 实现数据模型和数据库
3. 实现任务 CRUD 操作
4. 添加 UI 组件和交互
5. 实现通知和提醒功能
6. 添加数据导入导出功能
