# Window 类

## 概述
Window 类用于管理应用窗口，提供创建、控制和自定义窗口的功能。

## 方法

### open($url, $options = [])
打开一个新窗口。

#### 参数
- `$url` (string): 要加载的 URL
- `$options` (array): 窗口选项
  - `width` (int): 窗口宽度，默认 800
  - `height` (int): 窗口高度，默认 600
  - `title` (string): 窗口标题
  - `resizable` (bool): 是否可调整大小，默认 true
  - `fullscreen` (bool): 是否全屏，默认 false

#### 返回值
(int): 窗口 ID

#### 示例
```php
use Native\ThinkPHP\Facades\Window;

// 打开一个新窗口
$windowId = Window::open('/dashboard', [
    'width' => 1024,
    'height' => 768,
    'title' => '控制面板'
]);
```
```
